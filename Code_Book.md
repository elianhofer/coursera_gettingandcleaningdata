* `activity` - The name of the activity performed during the observations

## Activity Labels

* `WALKING` (corresponds to value `1`): subject was walking during the test
* `WALKING_UPSTAIRS` (corresponds to value `2`): subject was walking up a staircase during the test
* `WALKING_DOWNSTAIRS` (corresponds to value `3`): subject was walking down a staircase during the test
* `SITTING` (corresponds to value `4`): subject was sitting during the test
* `STANDING` (corresponds to value `5`): subject was standing during the test
* `LAYING` (corresponds to value `6`): subject was laying down during the test
